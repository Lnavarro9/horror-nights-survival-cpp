# 🎃 Halloween Horror Nights Survival (C++)

A C++ console-based simulation game where the player must survive a full night at a horror theme park by managing time, energy, fear, and strategic decisions.

Built to simulate real-time decision-making under pressure using C++ logic and state management.

---

## 🧠 Features

- Decision-based survival system  
- Multiple win/loss conditions  
- Random event system (jump scares, shortcuts, delays)  
- Risk vs reward decisions (skip lines vs fear increase)  

---

## 🎮 Gameplay

Players can:
- Enter haunted houses 👻  
- Ride attractions 🎢  
- Rest or eat 🍔  

Time progresses through the night:
- 7 PM: Enter park  
- 8 PM: Haunted House (+fear, -energy)  
- 10 PM: Ride break (-time lost)  
- 1 AM: Low energy… final push  

Each action affects:
- Fear level  
- Energy  
- Health  
- Time  

Random events add unpredictability to each run.
