# 🎃 Halloween Horror Nights Survival (C++)

A C++ console-based simulation game where the player must survive a full night at a horror theme park.

## 🧠 Features
- Player stats (health, fear, energy)
- Time-based gameplay (7 PM – 2 AM)
- Random scare events
- Decision-based survival system
- Multiple win/loss conditions

## 🎮 Gameplay
Players choose actions like:
- Enter haunted houses 👻
- Ride attractions 🎢
- Rest or eat 🍔

Each action affects:
- Fear level
- Energy
- Health
- Time

Random events add unpredictability to each run.

## 🏆 Objective
Survive until 2 AM without:
- Fear reaching 100
- Health reaching 0
- Energy running out

## 💻 Technologies
- C++
- Standard Library
- Console-based UI

## 🚀 How to Run
1. Open in Visual Studio
2. Build & Run
3. Follow menu prompts
